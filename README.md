# Roguecraft

A roguelike space shooter prototype built with Phaser 3, React, and Vite.

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open your browser to `http://localhost:3000`

### Build

```bash
npm run build
```

## Project Structure

```
roguecraft/
├── src/
│   ├── game/          # Phaser game logic
│   ├── components/    # React UI components
│   ├── scenes/        # Phaser scenes
│   └── main.tsx       # App entry point
├── public/            # Static assets
└── index.html         # HTML entry point
```
